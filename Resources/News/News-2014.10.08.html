Locator Buttons

October 8, 2014
Locator Buttons have arrived. These are current only set "manually" on my end - but some day I hope to have a mechanism for any one to add a locator.

The buttons show up in place of the old progress bar, which was never really needed since the cursor on the page serves that purpose.

Roll over the graphic on the main page and the locators will appear; you can hover over the buttons to see there description and what time the correspond to. Also, there are links on the right side that show the description and time.

Neither of these are available in fullscreen, but there are small matching tags along the top of the graphic showing where the locations are.

Implementation

The buttons and locator links (and the new tagged time and locator strings) are activated by javascript and are not activated if javascript it turned off.
